## Profolio_backend

---

How to start

---

```jsx
// 패키지 설치
yarn install

// 프로젝트 시작
yarn start
```

---

Commit Convention

---

| 작성 | 설명 |
| --- | --- |
| feat: 내용 | 기능 추가 |
| fix: 내용 | 버그 및 오류 수정 |
| docs: 내용 | 문서 수정 |

---

Branch Convention

---

| 머릿말 | 설명 |
| --- | --- |
| master | 최종 프로젝트 |
| develop | 기능 구현 |
| feature | 기능 단위 구현 |
